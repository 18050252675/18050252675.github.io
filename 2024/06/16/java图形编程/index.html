0614 图形编程

绘制菜单栏 JMenuBar

  • 菜单 JMenu 多个
    • JMenultem多个
  • 案例：
    • 游戏菜单：开始游戏、重新开始、退出游戏
    • 设置：动画开、动画关
    • 帮助：游戏说明、关于
  • 动作接听 ActionListener

监听目标:除了窗口之外，能够产生动作/形态上能够产生变化的组件

在适当位置(Application)创建自定义的监听对象

在适当位置（Application）给某些组件安装动作监听

为不同的组件设置不同的指令（字符串）

图形绘制

前提：要自定义面板

绘制组件：paintComponent()

  • 重写此方法后，虚拟机后独立开一条线程，不断调用此方法
  • 不要创建对象！！！
  • 不要此方法中写修改数据的代码

必备条件

  • 第一含必须保留，条用父类的此方法 super.paintComponent(g)
    • 保证所有的子容器都能够显示正常
  • 在此方法的最后最好写上自动更新和主动更新的代码

绘制内容

  • ​ 绘制图形—–矩形

    • 空心矩形
      • g.drawRect(x,y,w,h)
    • 实心矩形
      • g.fillRect(x,y,w,h)
    • 绘制图片
      • 图片类型：jpg png gif
      • 创建一个文件夹专门放图片
      • g.drawImage(图片对象，x,y,w,h,null);
